##How to run the bot
1. Edit the config.json file and fill out everything
2. run npminstall.bat

If the npminstall.bat fails here is a list of everything to install
```
    cleverbot-node
    cowsay
    discord.js
    discord.js-music
    figlet
    fs
    google
    urban
    ```
